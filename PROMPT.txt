在已有 Virtual Data Model 基础上新增 Column Resize 功能：用户拖拽列边界线（column resizer）调整列宽，列宽要在客户端本地持久化（localStorage），刷新页面后恢复上次列宽；列宽变更后整表要重新排版（数据切片范围会跟着变）。在已有 Hierarchical Column Headers 基础上新增 Column Sort 功能：单击列头调用 DataListener 重新请求排序后的数据切片（DataListener 需要接收 sort 字段名和 sort 方向两个参数并返回排序后的 data），表头视觉上要显示当前排序列和方向（小箭头图标），二次点击同列切换升降序。Column Resize 跟 Column Sort 联动时，列宽被重排后 hover 在列头的 resizer 仍能正常拖动，不能因为重排导致 resizer 错位。检查修复一下。
